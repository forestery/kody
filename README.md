# Kody — 双击即译，不知不觉记住单词

> 专为**大量阅读英文网站**的中文用户设计。安装即用，零配置，完全本地运行。

![Kody Screenshot](https://github.com/forestery/kody/blob/main/docs/screenshot.png)

---

## 它是什么

Kody 是一个 Chrome 浏览器插件。

**核心交互只有一个：双击任意英文单词，立刻弹出中文翻译。**

不需要划词，不需要右键，不需要切换页面。双击，看一眼，继续读。

但 Kody 做的不止翻译——它在后台悄悄帮你记住这些词。

---

## kody的特点

普通翻译插件帮你查词，用完就忘。Kody 的设计理念是**被动学习**：

- **重逢提醒**：再次遇到查过的词，卡片会显示"3天前你在 TechCrunch 遇到过这个词，当时的句子是…"——情景记忆比死背单词有效得多
- **路过复习**：页面加载后，右下角悄悄滑出一张小卡片，展示一个该复习的词，5秒后自动消失，无需任何操作
- **随机漫步**：加入学习的单词，在网页中出现时会有橙色下划线标注，hover 显示中文，遇到次数越多角标越大
- **词汇画像**：后台统计你翻译了多少不同的词，在恰当时机告诉你"你已翻译 50 个词，词汇量预计 CET-4 水平"

所有这些都在浏览网页的过程中自然发生，**不需要打开任何学习页面，不需要完成任务，不需要主动背诵**。

## 功能一览

| 功能 | 说明 |
|------|------|
| 🔍 即时翻译 | 双击任意英文单词，弹出翻译卡片，支持多义项（词性 + 释义） |
| 📍 重逢卡片 | 查过的词再次双击，显示上次遇到的时间、网站和例句 |
| 🎯 领域感知 | 两次遇到同一词均在科技/财经/学术等同类网站，额外提示 |
| ★ 加入学习 | 点一下，加入基于艾宾浩斯记忆曲线的复习计划 |
| 🔶 随机漫步 | 学过的词在网页中高亮，角标显示遇到次数 |
| 👻 路过复习 | 打开新页面时，右下角静默出现复习卡，5秒自动消失 |
| 📊 词汇画像 | 根据翻译词数动态推算词汇水平，达到里程碑时在卡片内提示 |
| 🌐 离线词典 | 内置 ECDICT 35万词，完全本地，无网络请求，保护隐私 |
| 🔄 API 兜底 | 词典里没有的词，自动调用 MyMemory 免费 API 翻译并缓存 |

## 安装方法

### 方法一：从源码安装（当前唯一方式）

Kody 目前尚未上架 Chrome Web Store，需要手动加载。

**第一步：下载源码**

```bash
git clone https://github.com/forestery/kody.git
```

或者直接下载 [ZIP 包](https://github.com/forestery/kody/archive/refs/heads/main.zip) 并解压。

**第二步：打开 Chrome 扩展管理页**

在地址栏输入：
```
chrome://extensions/
```

**第三步：开启开发者模式**

页面右上角，打开「开发者模式」开关。

**第四步：加载插件**

点击「加载已解压的扩展程序」，选择刚才 clone 或解压的 `word-translator` 文件夹（里面包含 `manifest.json` 的那一层）。

**第五步：完成**

浏览器工具栏出现 **K** 图标，即安装成功。

> ⚠️ 注意：手动加载的插件在浏览器更新后可能需要重新加载。如果插件失效，回到 `chrome://extensions/` 点击刷新按钮（↺）即可。

## 使用方法

### 基本翻译

在任意网页上，**双击**一个英文单词，翻译卡片立刻弹出。

- 单击 **✕** 或点击卡片外区域关闭
- 按 **Esc** 关闭
- 支持多义词，按词性分行显示

### 加入学习

翻译卡片底部有「★ 加入学习」按钮，点击后：

1. 这个词加入记忆曲线复习计划（1天→3天→7天→15天→30天→60天）
2. 当前页面的上下文例句自动保存
3. 页面中所有出现该词的位置加橙色下划线标注

### 重逢提醒

再次双击**已学过的词**，卡片会额外显示：

```
📍 3天前 · techcrunch.com
"The company's ephemeral messaging feature was..."
🎯 上次也是科技类，同领域巩固中
```

### 路过复习

打开新页面约 1 秒后，如果有到期复习的词，右下角会出现一张小卡片：

```
┌──────────────────────────┐
│ ephemeral  短暂的         │
│ "...ephemeral fame..."   │
│  ✓ 记得    ✗ 忘了        │
│ ▓▓▓▓▓▓░░░░ (5秒后消失)  │
└──────────────────────────┘
```

- 点「✓ 记得」：推进记忆阶段，下次更晚复习
- 点「✗ 忘了」：重置阶段，明天再来
- 什么都不点：5 秒后自动消失，不影响任何进度

### 查看学习记录

点击浏览器工具栏的 **K** 图标，右键选择「⚙️ Kody 设置」，打开设置页面。

设置页展示：
- 词汇画像（推算词汇水平）
- 学习中的单词列表，含遇到次数和每次的来源链接
- 可按最近遇到 / 字母 / 复习时间 / 遇到次数排序
- 可搜索、移除单词、重置数据

## 隐私说明

- **词典完全本地**：内置 ECDICT 35万词，查词不发送任何网络请求
- **学习数据本地存储**：所有数据保存在浏览器本地（`chrome.storage.local`），不上传任何服务器
- **API 兜底**：仅当本地词典找不到某个词时，才调用 [MyMemory](https://mymemory.translated.net/) 免费翻译 API（发送的只是单个单词，无上下文）。翻译结果会缓存到本地，同一个词只请求一次

## 技术栈

- Chrome Extension Manifest V3
- 纯原生 JavaScript，无框架依赖
- 词典数据：[ECDICT](https://github.com/skywind3000/ECDICT)（MIT License）
- 翻译 API 兜底：[MyMemory](https://mymemory.translated.net/)（免费，无需 Key）

### 项目结构

```
word-translator/
├── manifest.json          # MV3 配置
├── background.js          # Service Worker（右键菜单）
├── content.js             # 核心逻辑：翻译、学习、复习
├── content.css            # 注入样式
├── options.html           # 设置页面
├── options.js             # 设置页逻辑
├── icons/                 # 插件图标
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── wordlists/             # ECDICT 词典（按首字母分26个文件）
    ├── dict_a.json
    ├── dict_b.json
    └── ...
```

## 适合人群

- 有一定英语基础（CET-4 以上），但词汇量还足以理解英文原文的情形
- 有大量阅读英文网站的习惯或需要（技术文档、新闻、学术论文等）
- 不想主动背单词，希望在日常阅读中自然积累词汇，0心理学习负担
- 注重隐私，不想把阅读内容发送给第三方服务
